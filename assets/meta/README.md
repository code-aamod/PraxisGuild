# Assets guidelines

Structure:
- assets/source/   -> master editable files (Figma links, .ai)
- assets/exports/   -> production-ready assets (SVG, PNG, PDF)
- assets/meta/      -> LICENSES.md, README, changelog

Naming rules:
- logo.svg
- triad-logo-horizontal.svg
- rank-02-adept.svg
- icon-*.svg

Export tips:
- Prefer SVG for logos and badges.
- Provide @2x PNGs for raster fallbacks.
- Social OG image: 1200x630px.
